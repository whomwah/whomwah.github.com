This is a directory of software written by Duncan Robertson and is hosted on Github. You can see other stuff at code.whomwah.com

RadioAunty

RadioAunty is a Cocoa app that runs on your mac desktop and allows you to listen to BBC National Radio live and onDemand outside your web browser. It was written to aid my learning Cocoa and also to serve my own personal needs for an application of this type.

TellyBox

TellyBox is a Cocoa app that runs on your mac desktop and allows you to watch BBC National Tv live and onDemand outside your web browser. It was written to aid my learning of Cocoa and also to serve my own personal needs for an application of this type.

TweetMe

TweetMe is a Wordpress plugin that posts a tweet to Twitter when you publish a blog post.

RevCanonical

RevCanonical is a Wordpress plugin adds url shortening and rev=canonical support.

QLStephen

QLStephen is an Apple OSX QuickLook plugin that lets you view plain text files without a file extension.

rQRCode

rQRCode is a library for encoding QRCodes in Ruby. It has a simple interface with all the standard qrcode options. It was adapted from the Javascript library by Kazuhiko Arase. It is available as a gem or you can just download the source.

Whomwah Wordpress Theme

Whomwah Wordpress Theme is as you may expect the Wordpress theme used on my site whomwah.com that you can add to your Wordpress enabled site. It worked on my old 2.6 version and works fine on my new 2.7 install.

BBCProgrammes iPhone WebApp Experiment

This was an experiment in building an native iphone webapp using just Dashcode as the IDE. It uses the BBC Programmes restful API. You can drill down from tv and radio services and view day schedules.

Langauge Timothy

A Google Wave bot that removes bad language from waves.

BBC Now Next Chrome extension

This Google Chrome extension lets you see at-a-glance what's now on or is coming up on BBC Radio and Television. You can see the extension on the Chrome extension site
